Q-Sort Tool

Sort statements into a grid to represent a viewpoint, priority, or experience.

What is a Q-sort?

A Q-sort is a structured way of ranking statements (or “thought cards”) along a scale such as “most like me” to “least like me”. It’s often used in psychology and education to capture a person’s perspective in a form that can be discussed or analysed.

How can it be used?

  • Supervision / coaching: sort challenges from “most pressing” to “least pressing”.
  • Therapy / reflection: sort beliefs from “strongly true” to “strongly untrue”.
  • Teaching: sort concepts from “confident” to “not confident”.
  • Research: collect comparable ranked sorts across participants.

How to use this app

  1. Start a new Q-sort (or load one you saved earlier).
  2. Give it a title/topic (e.g., “My current stressors”).
  3. Add thought cards (positive / neutral / negative if you want that structure).
  4. Drag cards into the grid to rank them.
  5. Save as JSON to continue later, or export the grid as an image.
